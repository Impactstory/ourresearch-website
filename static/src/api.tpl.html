API and datasets

There are are two main ways to access oaDOI's data: the API and the datasets.

The API is much easier to use, so it's probably your best bet. Most applications will not have scale problems using the API, as it can handle 100k/day from your application, and average return time is <200ms. The datasets are quite large (90M and 19M rows) and so using them requires some extra work, but they are great for some large research projects. The API is free for both commercial and noncommercial use.

Whichever you're using, we recommend you subscribe to the mailing list in order to stay up-to-date when there are changes or new features.

Datasets

The datasets behind the oaDOI API are available for download as two big CSV files. They are free for use in non-commercial academic research. Contact us if you're interested in downloading them.

If you'd like to download the datasets for commercial use, check out our service-level agreement program.
