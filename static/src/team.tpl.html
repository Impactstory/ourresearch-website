Team

oaDOI is being built at Impactstory by Heather Piwowar and Jason Priem, funded by the Alfred P. Sloan foundation.

We'd like to thank all of the people who've worked on earlier projects (OA Census, Open Article Gauge, Dissemin, Cottage Labs, and the Open Access Button) for sharing ideas in conversations and open source code -- in particular DOAI for inspiring the DOI resolver part of this project. Thanks also to the data sources that make oaDOI possible.

All of the code behind oaDOI is open source on GitHub.

Questions or ideas? You can reach us at team@impactstory.org or @oadoi_org.
